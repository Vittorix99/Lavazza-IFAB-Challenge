================================================================================
  TROUBLESHOOTING
================================================================================

ERRORE: anthropic.AuthenticationError 401 (invalid x-api-key)
  Causa:  La ANTHROPIC_API_KEY in Doppler è invalida o scaduta.
  Fix:    Genera una nuova chiave su https://console.anthropic.com → API Keys
          poi esegui:
          doppler secrets set ANTHROPIC_API_KEY=sk-ant-api03-TUACHIAVE \
            --project lavazza-ifab --config dev

ERRORE: Authentication failed su MongoDB Atlas
  Causa:  Il tuo IP non è in whitelist oppure la password è scaduta.
  Fix IP: doppler run -- python scripts\atlas_whitelist_ip.py
  Fix pw: Atlas → Database Access → Edit user → Autogenerate password
          poi aggiorna su Doppler:
          doppler secrets set MONGODB_URI=mongodb+srv://user:NUOVAPASSWORD@...

ERRORE: Connection refused su Qdrant (localhost:6333)
  Causa:  Il container qdrant non è avviato.
  Fix:    cd docker && docker compose up -d qdrant

ERRORE: Doppler "project not configured"
  Fix:    cd C:\Users\giaco\Desktop\Lavazza-IFAB-Challenge-main
          doppler setup --project lavazza-ifab --config dev --no-interactive

ERRORE: ModuleNotFoundError (dipendenza Python mancante)
  Fix:    cd lavazza-coffee-agent
          .\.venv\Scripts\pip install -r requirements.txt

ERRORE: Docker "cannot connect to daemon"
  Causa:  Docker Desktop non è avviato.
  Fix:    Apri Docker Desktop e aspetta che sia pronto.

ERRORE: UnicodeEncodeError su scripts\atlas_whitelist_ip.py
  Causa:  Il terminale Windows non supporta il carattere ✓.
  Note:   L'operazione è comunque andata a buon fine — ignora l'errore.

n8n non mostra workflow:
  Devi importarli manualmente la prima volta:
  1. Apri http://localhost:5678
  2. Settings → Import workflow
  3. Importa i 13 sub-workflow da docker\local-files\workflows\split\
  4. Importa il master: docker\local-files\workflows\Lavazza-MASTER-RUN.json
  5. Aggiorna i workflowId nel master con i nuovi ID assegnati da n8n
  6. Re-inserisci le credenziali API in n8n → Credentials
     (OpenAI, NASA FIRMS, USDA, AISSTREAM)
